WP-DXP is a plugin to allow you to personalize the content on your WordPress site. 

== Description ==

WP-DXP is a plugin to allow you to modify the content on your WordPress site. 

The plugin has been rebranded and redeveloped as [PersonalizeWP](https://wordpress.org/plugins/personalizewp/) and can be found at https://wordpress.org/plugins/personalizewp/

PersonalizeWP is still available as a free plugin but also has a paid option for those that require more functionality. It is being actively developed and has a full roadmap that will deliver improved and updated features and integrations with other plugins and services. 

If you are interested in personalisation, we strongly recommend using [PersonalizeWP](https://wordpress.org/plugins/personalizewp/) instead of WP-DXP. 

= Support =

This plugin is no longer being actively developed or supported.
